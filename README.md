# Pyman
A free open source JavaScript Pac-Man Battle Royale clone.

## Usage

1. Install prerequisites:
```bash
sudo apt install nodejs npm
npm install
npm audit fix --force
sudo ufw allow 3000
sudo ufw enable
```
2. Run Node.js solution:
```
node .
```
3. Open solution: [http://localhost:3000/](http://localhost:3000/).
